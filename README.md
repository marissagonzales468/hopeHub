# HopeHub Prototype

HopeHub is an immersive resource and donation platform designed to help individuals facing homelessness in Stockton, CA. This package contains both mobile and web prototypes, along with design assets and documentation to serve as a plug-and-play solution.

## Project Structure

- **assets/**: Contains design assets including the logo, icons, and fonts.
- **mobile/**: A React Native project for the mobile prototype.
- **web/**: A ReactJS project for the web prototype.

## Getting Started

### For Mobile:
1. Navigate to the `mobile` folder.
2. Run `npm install` to install dependencies.
3. Run `npm start` to launch the project with Expo.
4. Use the Expo Go app to view on your mobile device or emulator.

### For Web:
1. Navigate to the `web` folder.
2. Run `npm install` to install dependencies.
3. Run `npm start` to launch the development server.
4. Open your browser at `http://localhost:3000` to view the app.
